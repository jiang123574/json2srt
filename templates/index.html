{% load static %} 剪映json转换为srt字幕 {# #}

使用剪映识别字幕后导出srt

使用方法

首先将视频导入到剪映电脑版本

点击识别字幕

识别完成后关掉剪映

找到draft.json文件

windows路径: C:\Users\用户名\AppData\Local\JianyingPro\User Data\Projects\com.lveditor.draft\****-***-***-***-***\draft.json

macOS路径: ~/Movies/JianyingPro/videocut/****-***-***-***-***/template.json

在此页面选择文件

点击开始转换即可开始下载

注意:有时候需要关掉下载器才能下载字幕

另外:转换完成后 up主习惯使用 ARCTIME 进行校对
